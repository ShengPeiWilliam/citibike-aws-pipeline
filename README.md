# Citi Bike AWS Pipeline

A data pipeline that ingests Jersey City Citi Bike trip data into AWS, built as an exercise in cloud data engineering with S3, Athena, and boto3.

## Motivation

Previous projects focused on statistical modeling, A/B testing, and ML, all working with data that was already clean and ready to analyze. This project steps upstream: building the infrastructure that makes that kind of analysis possible at scale.

The goal is to learn cloud data engineering by doing, moving from local notebooks to a production-style pipeline on AWS.
